# 🎤 Voice Control Guide - SnipX AI

## Overview
SnipX AI features comprehensive **voice control** that lets you navigate and control the entire website using voice commands.

## Getting Started

### Activating Voice Control
1. Click the **floating microphone button** (bottom-right corner)
2. Allow microphone access when prompted
3. When the button turns **red and pulsing**, voice control is active
4. Say any command clearly

### Deactivating Voice Control
- Click the microphone button again, OR
- Say **"stop listening"**

---

## Available Voice Commands

### 🧭 Navigation Commands
Navigate between different pages:

- **"Go home"** / "Home page" / "Take me home" → Navigate to homepage
- **"Go to features"** / "Features page" → Open Features/AI Studio page
- **"Go to help"** / "Help page" / "Support" → Open Help & Support
- **"Go to profile"** / "My profile" / "Account settings" → Open your profile
- **"Sign out"** / "Log out" → Sign out of your account

### 📹 Video Processing Commands
Control video operations (Features page):

- **"Upload video"** / "Select video" → Open file selector
- **"Generate subtitles"** / "Create subtitles" → Generate subtitles for uploaded video
- **"Generate thumbnail"** / "Create thumbnail" → Create AI-powered thumbnail
- **"Download video"** / "Save video" → Download processed video
- **"Download thumbnail"** / "Save thumbnail" → Download generated thumbnail

### 📄 Page Control Commands
Control the current page:

- **"Scroll down"** / "Page down" → Scroll down
- **"Scroll up"** / "Page up" → Scroll up
- **"Scroll to top"** / "Go to top" → Jump to top of page
- **"Scroll to bottom"** / "Go to bottom" → Jump to bottom of page

### ℹ️ Help Commands
Get assistance:

- **"What can you do"** / "Help me" / "Show commands" → Display available commands
- **"Stop listening"** / "Pause" → Deactivate voice control

## Tips for Best Results

### 🎯 Clear Speech
- Speak **clearly and naturally**
- Avoid shouting or whispering
- Minimize background noise

### 📱 Browser Support
Voice control works best on:
- ✅ **Chrome** (Desktop & Mobile)
- ✅ **Edge** (Desktop)
- ✅ **Safari** (Desktop & iOS)
- ❌ **Firefox** (Limited support)

### 🌐 Languages
- **Primary**: English (US)
- Commands work with natural variations
- Example: "Upload video" = "Select video" = "Choose video"

### 🔄 Real-time Feedback
- **Transcript box** appears showing what you said
- **Toast notifications** confirm command execution
- **Voice responses** provide audio feedback

## Troubleshooting

### Voice Control Not Working?
1. **Check microphone permission**
   - Browser should request access
   - Check browser settings if denied

2. **Use supported browser**
   - Best: Chrome, Edge, Safari
   - Avoid: Firefox

3. **Check microphone**
   - Test in system settings
   - Ensure not muted

### Commands Not Recognized?
1. **Speak more clearly**
   - Reduce background noise
   - Speak at normal pace

2. **Try alternative phrases**
   - "Upload video" instead of "Upload file"
   - "Go home" instead of "Homepage"

3. **Say "help me"** to see all commands

## Privacy & Security

### Your Voice Data
- ✅ **Processed locally** in your browser
- ✅ **Not stored** on our servers
- ✅ **Not transmitted** anywhere
- ✅ **Uses browser's built-in Speech API**

### Microphone Access
- ✅ **Only when activated** (red pulsing button)
- ✅ **Can be revoked** anytime in browser settings
- ✅ **Clear visual indicator** when listening
